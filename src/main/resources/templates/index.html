Desarrollo de Aplicaciones Web II

Este es un examen realizado con Spring security,thymeleaf,Spring Data JPA,MySql,SpringDevTools,Spring Web,lombok,Diseño con Bootstrap 5.0.2,Hover.css 2.3.1, animate.css 5.14.4 . y font-awesome.

¿Que se ha realizado?

Se hizo el desarrollo de un aplicación con dos respectivos roles (ADMIN,USER) los cuales estan asociadas con distintas funcionalidades con las que se puede interactuar con la BD de la aplicación web.

Catálogo

¿Como se puede utilizar?

Primero se debe realizar un inicio de sesión para habilitar las funcionalidades respectivas. En caso no se haga un inicio de sesión solo se podrá visualizar un listado de los productos sin opcion a comprar o realizar las operaciones CRUD.

Catálogo

¿Que pasa si se quiere acceder a un recurso no permitido?

En caso de que un usuario quiera acceder a un recurso,ruta o sección que no esta autorizado ,se le mostrará un vista que indicara que no puede ingreesar al recurso por ser prohibido para el rol en sesión.

¿Que funciones puede realizar como ADMIN?

  • Listar Productos
  • Comprar Producto
  • Detalle Producto
  • Agregar Producto
  • Actualizar Producto
  • Eliminar Producto

¿Que funciones puede realizar como USER?

  • Listar Productos
  • Detalle Producto
  • Comprar Producto
